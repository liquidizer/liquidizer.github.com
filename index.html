What is the Liquidizer

The Liquidizer is an interactive voting game. It is used to find a collective answer to a number of conflicting questions or demands. It differs from similar projects like Doodle or LimeSurvey. The Liquidizer can handle a much larger amount of conflicting questions, because finding optimal answers is made a collaborative experice. Rich visualizations of conflicting positions, trends, and matches allow for an indepth reflection of a groups own preferences.

Installation

On Ubuntu you can install the Liquidizer by the following commands: sudo apt-get install git-core maven2 gnuplot graphviz git clone git://github.com/liquidizer/liquidizer.git cd liquidizer mvn jetty:run After that a new Liquidizer instance will be accessible on http://localhost:8080.
